# assets/logo/

Đặt logo vào đây với ĐÚNG tên file:

| Tên file      | Dùng ở đâu                                  | Kích thước gợi ý |
|----------------|----------------------------------------------|-------------------|
| `logo.png`     | Màn hình vào phòng (Admin + Player), nền trong suốt | 512×512 (vuông), nền PNG trong suốt |

Nếu CHƯA có `logo.png`: game tự hiện logo chữ thay thế ("🐺 MA SÓI" trong khung tròn dát viền vàng) — không bị vỡ layout, không hiện icon ảnh lỗi.

Khi bạn thêm đúng file `logo.png` vào đây, logo ảnh sẽ tự thay thế logo chữ — không cần sửa code.
