# assets/ui/

Đặt các icon UI nhỏ (phase, trang trí) vào đây với ĐÚNG tên file:

| Tên file              | Dùng ở đâu                                  | Kích thước gợi ý |
|-------------------------|-----------------------------------------------|-------------------|
| `icon-night.png`        | Cạnh chữ "ĐÊM x" trên Phase Banner             | 64×64, nền trong suốt |
| `icon-day.png`          | Cạnh chữ "NGÀY x" trên Phase Banner            | 64×64, nền trong suốt |
| `icon-lobby.png`        | Cạnh chữ "Phòng chờ"                          | 64×64, nền trong suốt |
| `icon-ended.png`        | Cạnh chữ "KẾT THÚC"                           | 64×64, nền trong suốt |

Hiện tại Phase Banner đang dùng emoji (🌙 ☀️ 🛋️ 🏁) làm icon mặc định — luôn hiển thị đẹp, không cần ảnh. Nếu bạn thêm đúng tên file ở trên vào đây, icon ảnh sẽ tự hiện THÊM cạnh emoji (không thay thế emoji, để đảm bảo không bao giờ bị trống icon nếu ảnh lỗi/thiếu).

Nếu chưa có ảnh: không có gì hiện thêm, emoji vẫn hiển thị bình thường — không lỗi, không vỡ layout.
